# Mimesis Canvas

A practical worksheet for artifact-level Mimesis Engineering.

Use it before asking an LLM to create, rewrite, design, or judge something.

## Status

Public worksheet surface.

This repository contains canvas templates and examples for running a Mimesis
loop. It does not prove external adoption, benchmarked productivity, customer
outcomes, or universal improvement.
