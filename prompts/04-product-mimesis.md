# 04 Product Mimesis

```text
I am building this product:
[description]

Use product-level mimesis.

Compare it against:
[products]

Analyze:
1. first-screen promise
2. onboarding path
3. core action
4. repeated-use loop
5. trust and proof
6. pricing or value logic
7. failure and empty states

Then produce:
- product structure map
- missing loops
- risky claims
- smallest testable version
```
