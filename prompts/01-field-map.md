# 01 Field Map

```text
I am working on:
[describe the product, page, workflow, or argument]

Map the field before improving it.

Find or infer:
1. direct competitors
2. predecessor products or papers
3. adjacent alternatives
4. failed attempts
5. open-source implementations
6. user expectations
7. hidden costs
8. claims I should not make yet

Return:
- field map
- strongest artifacts to inspect next
- failure modes
- first proof artifact I should create
```
