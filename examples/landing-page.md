# Example: Landing Page Mimesis

Use this when a landing page feels generic.

## Inputs

- current landing page
- 5 strong pages in the same category
- 3 failed or overhyped pages

## Output

- first-screen claim boundary
- information hierarchy map
- CTA map
- trust device map
- visual grammar notes
- rewrite plan

## Rule

Do not copy the page.

Copy the reason the page works, then rebuild it under your own constraints.
