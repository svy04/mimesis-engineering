# Examples

Examples show where to start a Mimesis loop.

They are not proof that the method works universally.
They are small scenario guides for choosing inputs and outputs.

## How To Use

1. Pick the example closest to your weak artifact.
2. Pair it with a reference pack.
3. Fill the `.mimesis` artifact trail.
4. Run the validator if you are using this repository shape.
5. Run `npm run first-loop:demo` and `npm run audit:first-loop` if you want a generated local completed case before using your own artifact.

## Boundary

An example gives a starting pattern.
A case note gives proof of an actual transformation.
