# Example: Blog Homepage Mimesis

Use this when a blog homepage has strong ideas but too many competing entry points.

This scenario is informed by the `mimesis-canvas` blog example and the `mimesis-casebook` blog homepage case.

## Inputs

- current homepage
- concept page examples
- start-here page examples
- proof boundary page examples
- current screenshots if available

## Output

- first-screen thesis
- three reading paths or fewer
- method link
- case link
- proof boundary link
- visual or preview-image fixes
- next reader-feedback proof

## Seven-Day Version

Rewrite the first screen, add three reading paths, and publish one artifact-level essay or case.

## Boundary

This example does not prove reader conversion.
It prepares the homepage for clearer public testing.
