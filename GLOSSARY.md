# Glossary

## Mimesis Engineering

An artifact-first work system for AI-native builders.

## Artifact-first Builder

A builder who gives AI standards before asking for output.

## Roleplay Prompting

The old way.
It asks AI to pretend.

Example:

```text
You are a world-class expert.
```

## Artifact-level Mimesis

Using real expert artifacts as standards for AI-assisted work.

## Exemplar Context Pack

A structured pack of strong artifacts and their decomposed patterns.

## Reference Pack

A standard pack for a domain.
It defines what good artifacts usually do, what to inspect, what can transfer, and what must not be copied.

## Artifact Trail

The visible outputs of one Mimesis loop:

- Spec Lock
- Procedure Tree
- Artifact Brief
- Reference Set
- Structure Map
- Transformation Plan
- Improved Artifact
- Boundary Check
- Case Note
- Run Ledger

## Spec Lock

The file that fixes the goal, constraints, forbidden claims, and completion standard for a loop.

## Procedure Tree

The operating path for a loop:
Import, Distill, Capsule, Shard, Verify, Remember.

## Claim Boundary

The line between what can be claimed now and what needs more proof.

## Originality Boundary

The line between structure extraction and surface copying.

## Weak Mimesis

Imitating tone, style, layout, or surface.

## Strong Mimesis

Extracting judgment structure, proof loops, and transferable artifact patterns.

## Proof Surface

A public place where claims, evidence, limits, and next artifacts are visible.

## Case Note

A before/after record of what changed, what improved, and what remains unproven.

## Run Ledger

A record of actions, changed files, verification evidence, allowed claims, and next loop.

## Adapter

A documented way to run Mimesis inside a tool without changing the method.

## Plugin Shape

A future packaging target with explicit status and proof boundaries.
In v0.1, plugin shapes are not shipped plugins.

## Validator

A local check that inspects repository protocol consistency.
It does not prove external adoption or outcome improvement.
