# Quality Bar

Every public Mimesis artifact must pass these checks.

## 30-Second Clarity Test

Can a stranger understand what Mimesis is?

Expected answer:
Mimesis gives AI standards from real artifacts instead of asking it to roleplay expertise.

## First Action Test

Can a visitor do one thing immediately?

Expected action:
Bring one weak artifact and run the 5-minute first loop.

## Artifact Trail Test

Does the method create visible artifacts?

Expected artifacts:
Artifact Brief, Reference Set, Structure Map, Transformation Plan, Improved Artifact, Boundary Check, Case Note.

## Boundary Test

Does it clearly separate structure extraction from copying?

Expected line:
Weak Mimesis copies surface.
Strong Mimesis extracts structure.

## Proof Test

Does it avoid claims without evidence?

Expected line:
No proof, no claim.

## Fantasy Test

Does it feel like a future work system, not a generic prompt repo?

Expected feeling:
The visitor found a field manual, not a prompt pack.

## GitHub Test

Does it feel useful enough to star, fork, open an issue, or submit a case?

Allowed invitation:
Try the loop, submit a weak artifact, improve a reference pack, or report a boundary risk.

Forbidden invitation:
Fake stars, copied comments, artificial hype, invented proof.

## Protocol Validation Test

Can the local repository surface be checked without a hosted service?

Expected command:

```bash
node tools/validate-mimesis.mjs
```

Expected boundary:
The validator checks local protocol consistency.
It does not prove external adoption or productivity gains.

## Claim-Risk Test

Does the repository catch risky maturity language near the public surface?

Expected behavior:
Risky phrases such as external adoption, benchmarked productivity, shipped plugins, or verified integrations must appear only with nearby boundary language unless there is direct proof.

## Case Completeness Test

Does each case show both a change signal and a boundary signal?

Expected behavior:
Every case should make clear what changed, failed, shifted, or was transformed, and what remains bounded or unproven.
