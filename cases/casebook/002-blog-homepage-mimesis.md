# Case 002: Blog Homepage Mimesis

## Starting Artifact

The blog had strong concepts but too many front-page proof and stakeholder sections.

## Mimesis Lens

Good concept hubs reduce first-screen cognitive load, expose the main thesis, and move depth into reading paths.

## What Changed

- Removed performative section labels from the front.
- Kept the plain thesis: "AI가 만든 결과를 그냥 믿지 않기 위해 씁니다."
- Reduced the first path to method, case, boundary.
- Fixed preview image cropping.
- Improved terminal and code readability.

## What Remains Unproven

External reader conversion and audit demand.

## Next Proof Artifact
