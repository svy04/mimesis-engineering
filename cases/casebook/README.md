# Mimesis Casebook

Public cases for applying Mimesis Engineering.

## Status

Public case surface.

This repository contains case notes and case grammar for Mimesis Engineering.
It does not prove external adoption, benchmarked productivity, customer
outcomes, legal originality, or universal improvement.

Each case should show:

1. the starting artifact
2. the reference artifacts
3. knowledge structure
4. visual structure
5. user flow
6. what changed
7. what remains unproven
8. next proof artifact
