## How To Read Them

Use the casebook to study:

- starting artifact
- Mimesis lens
- what changed
- what remains unproven
- next proof artifact

Then use this repository to run the protocol:

- fill `.mimesis/`
- write a case note
- run `npm run validate`

## Boundary

These are public case resources.
They are not benchmark studies, customer proof, or universal evidence of effectiveness.
