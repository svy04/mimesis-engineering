# Cases

Cases are the public proof surface of Mimesis.

A case should show:

- before
- reference standard
- extracted structure
- after
- what improved
- what remains unproven

## Case Types

- transformation case: shows a before/after artifact change
- boundary case: shows where Mimesis becomes shallow copying
- positioning note: shows a strategic shift but does not prove outcomes
- public-source specimen: uses public materials as references without claiming customer proof

Use [docs/CASEBOOK-PROTOCOL.md](../docs/CASEBOOK-PROTOCOL.md) for the public case shape.

## Boundary

A case is evidence of a transformation.
It is not evidence of universal effectiveness.
