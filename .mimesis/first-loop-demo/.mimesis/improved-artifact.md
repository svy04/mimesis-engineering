# Mimesis First Loop

Bring one weak artifact.
Study one reference pack.
Transform the artifact under a visible proof boundary.

First action:

```bash
npm run first-loop:demo
npm run audit:first-loop
```

The generated demo shows the artifact brief, reference set, structure map, transformation plan, improved artifact, boundary check, case note, and run ledger.

## Why It Is Stronger

It is stronger because it replaces vague benefit language with a concrete action, named evidence, and explicit limits.

## What Remains Unproven

External adoption, benchmarked productivity, customer outcomes, legal originality, package publication, and real external proof remain unproven.
