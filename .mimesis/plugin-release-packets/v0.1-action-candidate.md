## Root Action Metadata

```yaml
name: Mimesis Protocol Check
description: Check a repository for a visible Mimesis .mimesis artifact trail.
inputs:
  working-directory:
    description: Repository or subdirectory to inspect.
    required: false
    default: "."
  command:
    description: Mimesis command to run. Use workspace:check for external repositories.
    required: false
    default: "workspace:check"
runs:
  using: composite
  steps:
    - name: Run Mimesis protocol check
      run: |
        case "${{ inputs.command }}" in
          workspace:check)
            node "$GITHUB_ACTION_PATH/bin/mimesis.mjs" workspace:check "${{ inputs.working-directory }}"
            ;;
          validate|release:check)
            cd "$GITHUB_ACTION_PATH"
            node bin/mimesis.mjs "${{ inputs.command }}"
            ;;
          *)
            echo "Unsupported Mimesis command: ${{ inputs.command }}" >&2
            exit 1
            ;;
        esac
      shell: bash
```

## Repository-Local Action Evidence

```yaml
name: Mimesis Release Check
description: Run the Mimesis local release preflight from a GitHub workflow.
inputs:
  workspace-check:
    description: Run the neighboring-repo ecosystem audit after the local release preflight.
    required: false
    default: "false"
runs:
  using: composite
  steps:
    - name: Run local release preflight
      if: ${{ inputs.workspace-check != 'true' }}
      run: npm run release:check
      shell: bash

    - name: Run workspace release preflight
      if: ${{ inputs.workspace-check == 'true' }}
      run: npm run release:check:workspace
      shell: bash
```

Run:

```bash
npm run audit:action
npm run workspace:check
```

The first command checks the root action metadata and boundary docs.
The second command checks the current repository's `.mimesis` protocol trail.
