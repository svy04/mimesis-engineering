# `.mimesis` Protocol

This folder shows the default file protocol for one Mimesis loop.
In this repository, it also records the v0.1 "Mimesis on Mimesis" run.

Copy it into a project when you want a visible artifact trail.

The files are intentionally plain Markdown.
They work with Codex, Claude Code, Gemini CLI, Cursor, or a human editor.
