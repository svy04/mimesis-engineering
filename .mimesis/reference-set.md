## Selection Rule

Use references for structure.
Do not use them as costumes.

## Source-First Check

- Local repo truth was checked before external references.
- Official docs and primary/original sources were preferred where concrete integration or provenance rules mattered.
- Secondary commentary was not used as proof.
- References are visible enough for review without implying endorsement.
