# Completion Audit

Status: current v0.1 completion matrix.

This audit prevents the repository from redefining completion around whatever already exists.
It separates finished local framework evidence from proof gates that still need a user decision, external artifact, or public release event.

## Completion Standard

The framework is complete enough for v0.1 only when:

- the method is understandable in 30 seconds
- a builder can run a first loop in 5 minutes
- the first loop has a generated local completed-demo case checked by workspace and case audits
- the AI-native framework manifest can be generated and audited before validation
- the framework manifest has a local schema contract checked before validation
- the spec index can be audited before local schema contracts are treated as discoverable AI-native framework inputs
- the README and quickstart activation surface can be audited before public claims
- the file protocol is visible and copyable
- reference provenance is visible
- reference packs are indexed as local source-first standards before source-first claims are checked
- proof and originality boundaries are explicit
- cases show before/after and boundary
- adapters and plugins have status labels that match evidence
- local validation can be run with one command
- one weak artifact can be turned into a started case workspace
- completed local case evidence can be checked before making a case claim
- permissioned external case intake can be reviewed before publication
- a reviewable permissioned-case fixture can be audited without claiming real external proof
- reviewed permissioned intake can create a started external case workspace
- reviewed schema-shaped proof intake records can create a started case workspace
- completed case evidence can be packaged as a bounded casebook candidate
- strong public claims can be checked through an evidence packet gate

- draft evidence packets can record a named reviewer decision without creating evidence

- local credential-pattern and risky-file scanning can run before publication claims
