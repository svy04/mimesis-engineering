# Superpowers Adapter

Status: local adapter contract, not execution proof.

The Superpowers adapter gives agents a short operating contract for combining:

- Superpowers process discipline
- Mimesis Engineering artifact standards

Use it when the operator wants a visible method rather than roleplay.

## Local Commands

Generate the adapter packet:

```bash
npm run adapter:superpowers
```

Audit the adapter surface:

```bash
npm run audit:superpowers-adapter
```

The generated packet is:

```text
.mimesis/adapter-packets/superpowers.md
```

## Method

Use this loop:

```text
Import -> Distill -> Capsule -> Shard -> Verify -> Remember
```

Superpowers governs the working discipline.
Mimesis governs the artifact trail.

## Boundary

This adapter does not install Superpowers.
It does not prove that a Superpowers skill ran.
It does not prove package publication, external adoption, benchmarked productivity, shipped plugins, legal originality, or external proof.
It only records a local contract for using Superpowers and Mimesis together.
