# Public Claim Pack

Status: local copy boundary guardrail.

`claim:pack` generates a public copy packet for Mimesis Engineering v0.1.
It turns the local framework evidence into allowed public claims, disallowed public claims, copy snippets, stop conditions, and evidence links.

## Command

```bash
npm run claim:pack
```

This writes:

```text
.mimesis/claim-packs/public-v0.1.md
```

Audit it with:

```bash
npm run audit:claim-pack
```

## What It Protects

Mimesis can sound bigger than the current proof.
This pack keeps public language inside the current local evidence:

- say what the framework contains
- say how to run the local checks
- cite the proof boundary
- name the remaining gaps
- block fake engagement, fake proof, and exaggerated maturity

## Allowed Public Claims

- Mimesis Engineering v0.1 is a Markdown-first, artifact-first AI-native work framework.
- It gives AI standards, not roles.
- It starts from one weak artifact and visible reference standards.
- It includes local file protocols, templates, reference packs, cases, prompts, adapters, plugin shapes, and validators.
- It can generate local handoff packets for proof, license, plugin, publish, gate, operator, and public-copy decisions.

## Disallowed Public Claims

- It does not prove external adoption.
- It does not prove benchmarked productivity.
- It does not choose a license.
- It does not publish to npm, GitHub Marketplace, or any package registry.
- It does not prove remote content freshness.
- It does not prove legal originality, customer outcomes, or production-grade security.

## Boundary

This guardrail does not publish, post, stage, commit, push, tag, create a pull request, create external proof, create engagement, choose a license, or certify marketing copy.

It is a local claim boundary before stronger public distribution.
